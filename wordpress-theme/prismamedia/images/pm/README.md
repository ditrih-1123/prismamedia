# Prime minister photos for "Trust in prime ministers" sidebar widget

Copy the following images from the Next.js project `public/images/` into this folder:

- Babish.png
- Fico.png
- Orban.png
- Muntianu.png
- Kobahidze.png
- Pashynyan.png

If you don't have these files, the sidebar will show broken images until you add them. You can use placeholder images or leave the folder empty and hide the avatar in the theme CSS.
